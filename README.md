# DARKNESS WITHIN

It is a 2d plateformer game with dark rougelike vibes something like a dungeon crawler .
It is made using godot game engine
In it you have fight mosters and collect coins , emeraleds , diamonds and clear levels

## FEATURES
1. Movements ------->
    Left movement
    Right movement
    Jump
    Attack1
    Attack2
    Dash
    Special Dash
    Jump Attack Side
    Jump Attack Up
    Jump Attack Down
   
2. Animations -------->
    Idle
    Running
    Dash
    Special Dash
    Attack1
    Attack2
    Jump Attack Side
    Jump Attack Up
    Jump Attack Down
    Fall
    Midair
    Jump
    Dash Attack
    Special Dash Attack
    Damage
    Death

3. Collectables --------------->
    Coins
    diamod
    Emerald




##CONTROLS

LEFT ARROW  =  LEFT MOVEMENT
RIGHT ARROW = RIGHT MOVEMENT
SPACE = JUMP
Q = DASH
Z = SPECIAL DASH
JUMP + ARROW UP = ATTACK UP 
JUMP + ARROW DOWN = ATTACK DOWN
Q +  LEFT CLICK MOUSE = DASH ATTACK

LEFT CLCIK MOUSE ONCE = ATTACK 1 ( LIGHT ATTACK )
LIFT CLICK MOUSE TWICE = ATTACK 2 (HEAVY ATTACK )
